Чат-бот на базе ИИ

О проекте

Данный проект представляет собой чат-бот на базе искусственного интеллекта, разработанный для обработки запросов и предоставления информации из подключенной базы знаний.

Бот интегрирован с платформой Telegram и использует современные технологии обработки естественного языка.

Технические характеристики

Мультиязычная поддержка

Система автоматически определяет язык входящего сообщения и формирует ответ на соответствующем языке. Базовый язык системы - русский.

Векторное хранилище

Реализовано векторное хранилище для эффективного поиска по базе знаний. Система способна находить релевантную информацию в загруженных документах.

API интеграции

Интеграция с Telegram через webhook API. Архитектура позволяет легко добавлять новые каналы коммуникации, включая WhatsApp.

Языковая модель

В основе системы - языковая модель GPT-4o-mini, обеспечивающая генерацию релевантных ответов с учетом контекста беседы.

Архитектура системы

Серверная часть

  • FastAPI для обработки webhook-запросов
  • PostgreSQL для хранения истории сообщений
  • SQLAlchemy ORM для работы с базой данных
  • Асинхронная обработка запросов

Компоненты системы

  • Модуль обработки сообщений
  • Векторное хранилище для поиска по документам
  • Интеграция с OpenAI API
  • Система логирования и мониторинга
